DESCRIPCION DEL PROYECTO:
Este proyecto consiste en el desarrollo de un plugin personalizado para osTicket, diseñado para gestionar el proceso de dictaminación de proyectos en entornos académicos.
El plugin permite automatizar la evaluación de propuestas mediante criterios predefinidos, centralizar el almacenamiento de dictaminaciones y facilitar la asignación de dictaminadores dentro del sistema. Además, mejora la trazabilidad, el seguimiento de evaluaciones y la eficiencia del flujo de trabajo.
Actualmente, el sistema se encuentra en uso dentro de una institución educativa 

EL PLUGIN SE PUEDE DESCARGAR Y UTILIZAR EN CUALQUIER ENTORNO DE UN SISTEMA osticket

PLUGIN: dictaminacion.phar
